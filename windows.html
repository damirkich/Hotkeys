Windows OS hotkeys

Ctrl Hotkeys

  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + X: Cut
  • Ctrl + Z: Undo
  • Ctrl + Y: Redo
  • Ctrl + A: Select All
  • Ctrl + F: Find
  • Ctrl + S: Save
  • Ctrl + P: Print
  • Ctrl + N: New window
  • Ctrl + Tab: Switch between tabs in a browser
  • Ctrl + D: Bookmark current page in a browser
  • Ctrl + Shift + Esc: Open task manager
  • Ctrl + Shift + Z: Undo the ctrl + z
  • Ctrl + Shift + T: Reopen last closed tab in a browser
